# Agrinho2026
OBJETIVO
Este projeto é um site que fala sobre o agronegócio e mostra a importância dele para a agricultura e agronegocio e para a economia. No site também são apresentados os benefícios da agricultura sustentável, algumas ações que ajudam a preservar o meio ambiente e como é possível produzir de forma mais consciente. O site foi produzido com html, Css e JavaScript.

O QUE CONTÉM NESSE SITE?
Neste site é notável a diversificação de cores, textos e parágrafos informativos, botões que difecionam a assuntos especificos e botões interativos. É UM site simples que tem apenas uma página, porém com tudo a oferecer ao leitor.

O QUE PROPORCIONA AO LEITOR?
Proporciona curiosidades, informações e uma dinamica divertida e cores vibrantes com o objetivo de deixar o leitor o máximo de tempo possível ao meu site.
